# Transfer Learning with Pretrained CNNs for Image Classification

##  Overview
This project demonstrates the effectiveness of transfer learning using pretrained Convolutional Neural Networks (CNNs) for image classification tasks.

The tutorial compares:
- A baseline CNN trained from scratch
- A pretrained model using feature extraction
- A fine-tuned pretrained model
- The impact of data augmentation

The Fashion-MNIST dataset is used to evaluate model performance.

---

##  Objectives
- Understand transfer learning in deep learning
- Compare different training strategies
- Analyse model performance using visualisations
- Evaluate training efficiency and generalisation

---

##  Dataset
- **Fashion-MNIST Dataset**
- 70,000 grayscale images (28x28 pixels)
- 10 clothing categories

---

##  Models Implemented
1. Baseline CNN (from scratch)
2. Transfer Learning (Feature Extraction)
3. Fine-Tuned Model
4. Transfer Learning with Data Augmentation
